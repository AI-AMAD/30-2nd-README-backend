## 초기기획 및 구현 목표
* 짧은 기간동안 기능구현에 집중해야하므로 사이트의 디자인과 기획만 클론

* 각 상품마다 시리즈가 있고 시리즈를 선택해서 장바구니에 담아 구매할 수 있는 기획으로 진행

* ## 개발기간  
    2022.03.14 ~ 2022.03.24  
    Sprint planning - 1 week

* ## 개발인원 및 파트

    * Front-end  
        김준영 - 소셜 로그인 페이지
